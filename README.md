# Email Threat Analysis Lab

## Overview
This project simulates a real-world phishing investigation where a suspicious email was analyzed to identify threats, extract indicators, and build detection strategies.

## Skills Demonstrated
- Email forensics
- IOC extraction
- Detection engineering
- Security reporting
